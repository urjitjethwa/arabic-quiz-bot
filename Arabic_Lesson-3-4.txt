Grammar + Object & Possessive pronouns.
Possessive pronouns (my, your, his, her, our, their) are used to indicate ownership of something. In
Arabic, as with object pronouns, these take the form of suffixes; they are attached to the noun that owns,
The object and possessive pronouns suffixes are exactly same except for the first person singular
1 For { my } add suffix ( ee ) with the noun. Kitaab-ee
2 For { our } add suffix (naa ) in the end of the noun. kitaab-naa
3 For { your (m) } add suffix ( k ) with the noun. kitaab-ak
4 For { your (f) } add suffix (ch ) in the end of the noun kitaab-ach
5 For { your (group) } add suffix ( kom ) in the end of the noun kitaab-kom
6 ~=For {his } add suffix (h ) in the end of the noun Kitaab-ah
7 For { her } add suffix ( haa ) in the end of the noun. Kitaaba-haa
8 For { their } add suffix ( hom ) in the end of the noun. kitaab-hom
